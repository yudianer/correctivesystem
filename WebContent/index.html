ELISA IE Room (Tibetan) Instruction

Thank you for participating Tibetan Room named entity tagging competition!

1. Overview
The interface has two components. The top panel contains test sentences from one article. You will tag tokens that are person names, geopolitical entity names or organization names. The definition of name types is at https://www.ldc.upenn.edu/sites/www.ldc.upenn.edu/files/english-entities-guidelines-v5.6.6.pdf. The bottom panel shows information including Tibetan dictionary, related sentences with annotation and gazetteers to help you decide which token to tag.

2. Tagging
In the top section, click one token, then in the input box below the sentence, you should type type a name as 'per (person)', 'gpe (geo-political entity)' or 'org (organization)'. Any content other than these three tags won't be recorded. After you enter the name type and press enter, a tag will show beside the token. Red is 'gpe', blue is 'per' and green is 'org'.

3. Information panel
When you click one token in the top panel, some related information will be shown in bottom panel. The Tibetan introduction is a brief introduction about written Tibetan language. Each dash box shows Chinese and English translation of selected token. The related words to the selected token in gazetteers will also be shown in the dash box. You can click gazetteer links to get more information. The sentences in the entire corpus that contain the selected token are listed below. The token with grey background is the selected token. The tokens with dash underline have Chinese and English translation. You can hover on these tokens to see the translation.

Tibetan Instruction

The written Tibetan language has 30 consonants, 4 written vowels, and 1 inherent (unwritten) vowel. Spoken Tibetan, however, contains at least 35 consonants and 9 spoken vowels, and the standard pronunciation of written Tibetan uses all 44 of these.
